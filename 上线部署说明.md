# 首浴堂代理订货系统上线部署说明

这个版本已经预留真实联网接口：

- 代理申请会写入 Supabase，并在总台通知中出现
- 总台可以创建代理账号，并生成账号密码
- 代理账号可以登录
- 代理下单会写入订单表
- 总台填写物流后，代理端订单里可以查看
- 商品上架会写入产品表

## 1. 创建 Supabase 项目

1. 打开 Supabase 官网并创建项目
2. 进入项目后台的 SQL Editor
3. 把 `supabase-schema.sql` 整份复制进去执行

执行后会创建这些表：

- `merchant_accounts`：代理/门店账号
- `products`：拿货商品
- `orders`：拿货订单和物流
- `applications`：代理申请/开通账号申请
- `notices`：总台通知

系统会预置一个测试代理账号：

- 账号：`STORE-001`
- 密码：`123456`

## 2. 准备 Supabase 密钥

在 Supabase 项目里找到：

- Project URL
- service_role key

注意：`service_role key` 不能放在网页里，只能放在 Vercel 环境变量里。

## 3. 发布到 Vercel

把 `outputs` 这个文件夹作为网站项目上传到 Vercel。

Vercel 里需要配置这些 Environment Variables：

```text
SUPABASE_URL=你的 Supabase Project URL
SUPABASE_SERVICE_ROLE_KEY=你的 Supabase service_role key
SESSION_SECRET=随便生成一串很长的随机密码
ADMIN_LOGIN=admin
ADMIN_PASSWORD=你给总台员工用的后台密码
```

## 4. 登录方式

总台后台：

- 打开 `/admin.html`
- 输入后台账号和后台密码
- 后台账号默认是 `admin`
- 后台密码是你在 Vercel 里设置的 `ADMIN_PASSWORD`

代理端：

- 打开 `/login.html`
- 测试账号：`STORE-001`
- 测试密码：`123456`

## 5. 第一版上线后的日常流程

1. 新门店在登录页提交代理申请
2. 总台后台登录后，在通知框看到申请
3. 总台创建商户账号
4. 系统显示新账号和初始密码
5. 代理商用账号登录
6. 代理商选择商品并提交拿货单
7. 总台在订单与发货里看到订单
8. 总台填写物流公司和物流单号
9. 代理商在“我的订单”里查看物流
